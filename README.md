# Gator

Gator is a command-line RSS aggregator that allows users to register, follow feeds, scrape posts, and browse recent content from their terminal.

Commands Available:
- register
- login
- addfeed
- follow
- following
- browse
- agg

## Requirements

Before running Gator, make sure you have the following installed:

- Node.js (LTS)
- npm
- PostgreSQL
- Git
